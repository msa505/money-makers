# AI Website Cloner 变现分析

## 一句话定位
一个 Next.js 模板项目，用 AI 编码代理（Claude Code / Opus）把任意网站反向工程成干净的、可部署的 Next.js 代码，适合快速复制落地页、营销站、SaaS 官网。

## 商业模式 / 收费方式

| 收费点 | 说明 |
|--------|------|
| **按次克隆服务** | 用户给 URL，你跑 AI 生成代码，交付源码包，按复杂度收费 |
| **订阅会员** | 月付无限次克隆 / 无限次修改 / 使用模板库 |
| **模板市场** | 把热门网站风格做成模板，卖给不会写代码的人 |
| **网站建设代运营** | 克隆后帮客户部署、改文案、SEO、维护，按月收费 |
| **卖工具本身** | 把模板改成低代码建站工具，卖给设计师/小老板 |

## 核心功能
- `/clone-website` 命令：AI 自动分析目标网站
- 提取设计 token（颜色、字体、间距）和 assets（图片、图标）
- 生成组件规范和并行构建任务
- 输出 Next.js + React + Tailwind 代码

## 变现路径（老王可走的路）

1. **AI 克隆接单服务**
   - 在朋友圈/小红书/独立站群里吆喝："给 URL，1 小时出可部署网站"
   - 按页收费：单页 ¥99-299，多页 ¥500-2000
   - 适合帮小老板、个体户、副业党快速复制竞品网站

2. **垂直模板站**
   - 用模板克隆 100 个热门行业落地页（电商、教育、餐饮、SaaS）
   - 卖模板：¥49-99 一个，或 ¥199 会员打包
   - 部署：直接挂在 Vercel/Cloudflare Pages 上

3. **SaaS 官网快速交付**
   - 接单帮初创公司做官网/落地页，用 AI 克隆加速交付
   - 结合你的设计/文案能力，卖 ¥2000-8000 一个项目

4. **做成在线工具**
   - 把 Claude Code 流程封装成网页：用户输入 URL → 付费 → 收到 ZIP 源码
   - 但核心是 Claude API 调用成本，需要算好定价

## 技术栈
- Next.js 15 + React + TypeScript
- Tailwind CSS + shadcn/ui
- Anthropic Claude API（核心驱动）
- Docker 支持（本地开发）

## 风险 / 注意点
- **版权风险最大**：克隆别人网站的外观、文案、图片可能侵权，交付时要替换素材和文案
- **AI 质量不稳定**：复杂交互、动画、响应式可能生成不完美，需要人工修
- **Claude API 成本**：Opus 调用一次克隆成本可能几美元，定价要覆盖成本
- **这是模板，不是成品**：需要你自己做产品化、计费、交付流程
- **MIT 许可证**：可商用，无传染性，最自由

## 一句话总结
**AI Website Cloner 是快速做网站复制生意的模板，变现靠接单和卖模板，但版权和 AI 质量是两个最大坑，必须人工兜底。**
